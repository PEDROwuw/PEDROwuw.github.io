HTML

HTML según sus siglas hipertext markup lenguage (lenguaje marcado de hipertexto) es un lenguaje de programación, está escrito en códigos que puede leerse sin que sea necesario hacer la prueba de escritorio, en pocas palabras, es una página web donde el texto está marcado con códigos que dan las órdenes al navegador sobre como plasmar el texto.

Estructura de HTML

La estructura de HTML es el procedimiento básico que seguir en la creación de una página web, para el entendimiento de este se debe conocer sobre las etiquetas ya mencionadas anteriormente:

Ante todo, se debe designar la apertura el lenguaje utilizando con la etiqueta “< html >” dentro de esta etiqueta debe ir todo el contenido y código de la página y al término de la elaboración se debe cerrar la etiqueta de esta manera “< /html >”.

< html >
Código de la página
< /html >

A continuación, se agrega la etiqueta “< head >” que es la encargada de ser la cabecera de la página y dentro de esta va otra etiqueta “< title >” que es el principal elemento de la cabecera; donde pondremos el título de nuestra página web, una vez agregado el titulo se cierra la etiqueta “” y seguidamente la etiqueta “”.

< html >
< head >
< title > “Título de la Página” < /title >
< /head >
Resto del código de la página
< /html >

Posteriormente se agrega la etiqueta “< body >”, dentro de esta pondremos todo el contenido que queramos en nuestra página y al terminar de hacerlo se debe de cerrar la etiqueta “< /body >” y seguidamente va el cierre de la página.

< html >
< head >
< title > “Título de la Página” < /title >
< /head >
< body >
Contenido de la página
< /body>
< /html >

Con esta guía se resume la estructura básica en el lenguaje HTML; como mención a recordar en el lenguaje html básico no afecta si las etiquetas se escriben tanto en mayúscula como en minúscula, pero en todo momento deben de estar correctamente escritas.

ENCABEZADOS Y TÍTULOS

Los encabezados HTML son etiquetas que se utilizan para definir los títulos y subtítulos de una página. Se pueden definir con la etiqueta < Hn > donde “n” es un número entre 1 y 6 que indica la jerarquía de dicha cabecera. Con esta etiqueta podemos implementar seis niveles de encabezado en el documento: < h1 > es con el tamaño de letra mayor y < h6 > con el tamaño más pequeño.

Etiqueta de título H1: El < h1 > es la etiqueta de mayor tamaño y se utiliza principalmente para los títulos de una página web.

Etiqueta de subtítulos H2: El < h2 > es la etiqueta que en HTML permite identificar los subtítulos. Normalmente se utilizan para producir un corte en el contenido.

Etiquetas h3, h4, h5 y h6: Las etiquetas < h3 >, < h4 >, < h5 > y < h6 > son utilizados para identificar subtítulos de menor jerarquía o bien, para los párrafos que se encuentran presentes dentro de secciones de < h2 > y permite ser minucioso con el contenido de nuestra página web. Hay que tener en cuenta que la etiqueta < h > podemos escribirla tanto en mayúsculas como en minúsculas. Es lo mismo escribir < h1 > que < H1 > aunque lo convencional es hacerlo en minúsculas. Además, no hay que olvidar que la etiqueta < h > tiene su etiqueta de cierre < \h > que ayuda a que no se mezclen los títulos y subtítulos y la página sea más ordenada.

Párrafos

Para los párrafos hay una etiqueta simple < p > que se emplean para aparcar un párrafo de texto y lleva su etiqueta de cierre < \p > la cual es opcional. Un párrafo comienza en una línea nueva y el navegador suele dejar un espacio en blanco extra antes del párrafo. Eso quiere decir que ya no es necesaria la utilización de otras etiquetas de separación de línea.

Alinear Texto

Para realizar la alineación de cualquier contenido a ser mostrado en una página web se utiliza el atributo “aling” (alineación), este atributo va acompañado del valor de la alineación el cual se quiere mostrar, por lo tanto puede tener uno de los siguientes valores.

“Right” alineación a la derecha. Ej: < p aling=”right”> < h1 aling= ”right”>
“Left” alineación a la izquierda. Ej: < p aling=”letf” > < h2 aling=”left”>
“Justify” alineación justificarr. Ej: < p aling=”justify”> < h4 aling=”justify” >
“Center” alineación para centrar. Ej: < p aling=”center” > < h5 aling=”center” >

El atributo “aling” se puede utilizar con cualquier etiqueta que cierre o introduzca un contenido. Cabe resaltar que la introducción de texto la alineación por defecto es hacía la izquierda, es decir, es irrelevante utilizarlo.

Lista

El lenguaje HTML nos ofrece opción de crear listas, estas pueden ser ordenadas, no ordenadas, alfabéticas, anidadas ylistas de glosario ofreciéndonos así una forma más organizada al momento de elaborar una página y también dando una apariencia más agradable.

Lista no ordenada

La etiqueta a utilizar es “< ul >” y para el contenido de las listas se usa la etiqueta “< li >” esta no es necesario cerrar ya que no afecta al código,una vez agregado todo en la lista se cierra la etiqueta “< /ul >”, esta etique posee un atributo "type" permite cambiar el elemento gráfico de la lista, y puden ser estos valores:
- circle: un círculo con el centro vacío
- disc: un círculo solido
- square: un cuadrado

Ejemplo
< ul type="square" >
< li > Contenido de la lista
< li > Contenido de la lista
< li > Contenido de la lista
< /ul >
Así se muestra en la página
  • Contenido de la lista
  • Contenido de la lista
  • Contenido de la lista
Listas Ordenadas

Para las listas ordenas utilizamos la misma estructura con la diferencia de la primera etiqueta que remplazamos con la etiqueta < ol >, esta etiqueta posee un atributo "type"; el atributo "type" indica el tipo de numeración de los elementos de la lista, y los valores de este pueden ser:

- A: indica una numeración con letras (A, B, C).
- a: indica una numeración con letras minúsculas (a, b, c).
- I: indica una numeración con números romanos en mayúscula (I, II, III, IV).
- i: indica una numeración con números romanos en minúscula (i, ii, iii. iv).
Ejemplo 1
< ol type="A">
< li > Contenido de la lista
< li >Contenido de la lista
< li >Contenido de la lista
< /ol >
Así se muestra en la página:
  1. Contenido de la lista
  2. Contenido de la lista
  3. Contenido de la lista

    Enlaces

    Los enlaces hipertexto ofrecen la posibilidad de pulsar sobre ellas para ir a otras páginas. Los enlaces se clasifican en dos tipos, los enlaces internos o locales y los externos:

    Los enlaces internos enlazan las páginas que componen un mismo sitio web. Por esta razón sólo será necesario indicar en nuestro código el nombre del archivo donde está la página que queremos enlazar y el directorio en el que se encuentra.

    Los enlaces externos permiten saltar de una página hasta otra, al crear un enlace externo es necesario indicar la dirección completa de internet (URL) de la página que queremos enlazar.

    Para colocar un enlace en la página se utiliza la etiqueta < a >...< /a >, con el atributo "href" que sirve para indicar cual es la página a la que se va a saltar:

    < a href="dirección de URL" >
    Texto que será sensible (hipertexto)
    < /a >
    En las páginas en otros directorios debemos indicar el directorio donde está:
    También puedes conocer mis hobbies

    Para separar los directorios se usa el símbolo "/", si fuese necesarios separar varios directorios. Los enlaces externos se cambia el valor del atributo "href" por la dirección completa de la otra página web, imagen o cualquier otro archivo

    Ancla

    Un ancla es una marca en un punto determinado de la página, las anclas es un enlace a un punto concreto de una pagina que desees. Para crear un ancla utilizamos la etiqueta < a >...< /a > y el atributo "name", el valor del atributo debe ir donde queremos que sea en ancla (este no debe tener espacios, acentos o caracteres extraños, solo guiones) y para cerrar el atributo con su valor se utiliza el simbolo"/" antes de cerrar la apertura de la etiqueta

    < a name="nombreAncla"/ > Inicio< /a >

    Para disponer de un ancla en un enlace se utiliza la etiqueta y el atributo de enlace "href" y en el valor utilizamos entre comillas y el símbolo "#" (el valor del ancla tiene que estar escrito exactamente igual al valor del ancla):

    < a > href="#nombreAncla"> Inicio < /a >

    Formato de Fondo
    El formato de fondo en HTML lo utilizamos para establecer el color o la imagen de fondo de una página web. Se puede establecer el color de fondo dentro de la etiqueta < body >utilizando el atributo "bgcolor" y el valor empezando con el símbolo"#" seguido del el color deseado en formato hexadecimal. Ejemplo: < body bgcolor ="#9478bb".
    Para agregar una imagen como fondo de la página se utilizamos el tributo “background” dentro de la etiqueta < body > , y el valor tendrá la dirección y el nombre de imagen que se utilizara como fondo. Este atributo tiene prioridad sobre el color de fondo, también hay que tener en cuenta que si la imagen no tiene el tamaño de la página mostrada en la pantalla esta se rellenara en cascada. El formato de fondo en HTML también permite ajustar la posición de la imagen de fondo utilizando la propiedad “background-position”, repetir la imagen de fondo utilizando la propiedad “background-repeat”, y ajustar el tamaño utilizando la propiedad “background-size”.
    Formularios
    ¿Qué es un formulario? Es una parte de la página conocida como caja con el propósito de que el usuario introduzca cualquier tipo de datos estructurados (nombres, apellidos, dirección, etc.)

    ¿Cómo crear un formulario?
    Para crear un formulario utilizamos la etiqueta principal < form > < /form > , dentro va la etiqueta < input > con el atributo "type" y el valor del atributo pueden ser; "text", "radio" o "submit".
    < input type=”text” > este sirve para una línea de texto. este mismo puede ir acompañado del atributo "value" que sirve para agregarle un valor en la caja de texto, ejemplo:
    < input type=”text” value="escribe tu nombre" > =>
    Pero luego el usuario deberá borrarlo en la pagina para escribir. para evitar esto existe otro atributo llamado "placeholder" el cual nos permite hacer lo mismo pero escribir en la caja de texto sin necesidad de borrar el valor ya escrito.
    < input type=”radio” > sirve para seleccionar varias opciones.
    < input type=”submit” > este sirve para agregar el botón de enviar en el formulario.
    La etiqueta < label > sirve para agregar un nombre a la etiqueta < input >. Ejemplo;
    < label > Nombre< /label >.=> . Otra etiqueta es < select > que es una etiqueta de selectores, dentro de ella introducimos la etiqueta < option > como su nombre lo indica sirve para poner las diferentes opciones por ejemplo; colores, números, nombres, etc.
    La etiqueta < textarea > sirve para crear un cuadro donde el usuario puede escribir lo que desee esta etiqueta lleva dos atributos "rows" y "cols" significan filas y columnas que el programador puede ajustar la cantidad que desee.
    Ejemplo:
    < html >
    < head >
    < title >Browser< /title >
    < /head >
    < body >
    < form >
    < label for="nombre" >Nombre< /label >
    < input type="text" id="nombre" name="name" > < /input>
    < label for="nombre" >Apellido< /label >
    < input type="text" id="apellido" name="name" > < /input >
    < label for=”color” >Elige un Color< /label >
    < select id=”color” name=”color” >
    < option >rojo< /option >
    < option >verde< /option >
    < option >amarillo< /option >
    < /form >
    < /body >
    < /html >
